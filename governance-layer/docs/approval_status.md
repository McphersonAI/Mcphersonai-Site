# Approval Statuses

Draft → Needs Review → Approved for Internal Use → Approved for Pilot.
Terminal/holding states: Blocked, Retired.

Only "Approved for Pilot" assets are eligible for live use, and Tier 3/4
assets additionally require the Pilot Readiness Checklist with recorded
Blake approval. "Blocked" means do not use until the blocking issue is
resolved and re-reviewed. "Retired" assets stay in the registry for history.
