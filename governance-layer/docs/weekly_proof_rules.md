# Weekly Proof Rules

- No fake customer quotes.
- No inflated ROI. Unverified numbers must be marked "estimate" — the
  validator rejects `time_saved_estimate` values that aren't.
- No identifiable client info unless explicitly approved.
- No confidential financial details.
- Private proof (full record) and public proof (sanitized case study) are
  separate artifacts; the public version is generated by stripping private
  fields, never by editing the private one in place.
