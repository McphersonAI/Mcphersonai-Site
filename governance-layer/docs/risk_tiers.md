# Risk Tiers

- **Tier 0 — Draft only.** Documents and placeholders. Cannot act.
- **Tier 1 — Internal tool.** Used by Blake only, fake/internal data.
- **Tier 2 — Assisted workflow.** Agent drafts, human executes everything.
- **Tier 3 — Live agent.** Agent acts within a scoped pilot. Requires full
  readiness checklist + Blake approval.
- **Tier 4 — External action agent.** Agent takes outbound/external actions.
  Highest bar: everything Tier 3 requires plus per-action approval design.

Tier assignment is recorded in the registry and re-checked at every review.
A tier can only move up after the readiness gate; it can move down anytime.
