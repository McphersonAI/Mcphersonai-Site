# 05_langfuse_observability — Langfuse Observability

## What belongs here

Langfuse setup notes, Docker stack verification, fake trace verification records, metadata-only mode confirmation, NullTrace/fallback test results.

## What must NOT be stored here

Langfuse credentials or .env files (local-only, never committed); raw trace content containing client/operator text.

## Reminder

No secrets, no `.env` files, and no real client data in any folder that could reach GitHub. Real pilot folders stay local/private.
