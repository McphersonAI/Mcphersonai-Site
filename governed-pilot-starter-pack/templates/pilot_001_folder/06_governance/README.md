# 06_governance — Governance

## What belongs here

Governance registration records: pilot registration, agent registration, module registrations, prompt version, risk tier assignment, approval status, allowed/forbidden tools list, registry exports.

## What must NOT be stored here

Unregistered assets in use; edits to registry exports after the fact (re-export instead).

## Reminder

No secrets, no `.env` files, and no real client data in any folder that could reach GitHub. Real pilot folders stay local/private.
