# 03_pilot_scope — Pilot Scope

## What belongs here

The approved pilot scope: what the pilot covers, what it explicitly excludes, success criteria, the 3x caught/avoided dollars target, and the scope approval record.

## What must NOT be stored here

Unapproved scope drafts presented as final; scope changes without a dated revision note.

## Reminder

No secrets, no `.env` files, and no real client data in any folder that could reach GitHub. Real pilot folders stay local/private.
