# 11_weekly_proof — Weekly Proof

## What belongs here

Completed weekly proof reviews (one per week of the live pilot), caught/avoided dollar records, and the running 3x-cost tally.

## What must NOT be stored here

Inflated or estimated-as-fact numbers — every proof event must trace to a logged SQLite proof event or a documented observation.

## Reminder

No secrets, no `.env` files, and no real client data in any folder that could reach GitHub. Real pilot folders stay local/private.
