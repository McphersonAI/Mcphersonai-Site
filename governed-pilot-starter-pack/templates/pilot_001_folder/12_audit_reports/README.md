# 12_audit_reports — Audit Reports

## What belongs here

Claude Code audit reports, dry-run evidence, starter pack verification output, and any module audit references for this pilot.

## What must NOT be stored here

Edited audit output; audits are preserved as delivered, with responses recorded separately.

## Reminder

No secrets, no `.env` files, and no real client data in any folder that could reach GitHub. Real pilot folders stay local/private.
