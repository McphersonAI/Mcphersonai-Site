# 13_go_live_approval — Go Live Approval

## What belongs here

The completed, signed go-live approval record (templates/go_live_approval_blank.md), plus any reactivation approvals after rollback.

## What must NOT be stored here

Blank or pre-checked approval forms presented as complete; verbal-only approvals — approval must be written and dated.

## Reminder

No secrets, no `.env` files, and no real client data in any folder that could reach GitHub. Real pilot folders stay local/private.
