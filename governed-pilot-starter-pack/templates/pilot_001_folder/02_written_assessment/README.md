# 02_written_assessment — Written Assessment

## What belongs here

The written assessment delivered to the operator: final version, delivery date, and any operator feedback notes.

## What must NOT be stored here

Draft versions with unredacted third-party data; secrets; anything not yet approved for the operator to see should be clearly marked DRAFT.

## Reminder

No secrets, no `.env` files, and no real client data in any folder that could reach GitHub. Real pilot folders stay local/private.
