# 04_sqlite_memory — Sqlite Memory

## What belongs here

The approved SQLite Memory Layer copy for this pilot, its config (non-secret), initialization records, and SQLite setup checklist results.

## What must NOT be stored here

The live database file in GitHub; real client data during test mode; secrets or credentials; backups (those go to 09_backups/).

## Reminder

No secrets, no `.env` files, and no real client data in any folder that could reach GitHub. Real pilot folders stay local/private.
