# 07_kill_switches — Kill Switches

## What belongs here

Kill switch default verification, kill switch test records, human-only mode test records, and the date each was last tested.

## What must NOT be stored here

Untested 'assumed working' claims — every entry here must reference an actual test run.

## Reminder

No secrets, no `.env` files, and no real client data in any folder that could reach GitHub. Real pilot folders stay local/private.
