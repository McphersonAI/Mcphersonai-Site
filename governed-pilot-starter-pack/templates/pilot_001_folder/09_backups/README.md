# 09_backups — Backups

## What belongs here

SQLite backups, backup verification records, and restore test records. First backup is created before go-live (deployment order step 13).

## What must NOT be stored here

Anything destined for GitHub — backups never leave this folder and never reach a repo; unencrypted backups containing real client data on shared machines.

## Reminder

No secrets, no `.env` files, and no real client data in any folder that could reach GitHub. Real pilot folders stay local/private.
