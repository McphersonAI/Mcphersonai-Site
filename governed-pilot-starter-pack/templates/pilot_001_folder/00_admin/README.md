# 00_admin — Admin

## What belongs here

Pilot administration: contact sheet (operator name/role only as approved), signed agreement reference numbers, pilot ID, key dates, scope summary, communication log.

## What must NOT be stored here

Full signed agreements with personal data (keep in your private document system and reference here), payment details, secrets of any kind.

## Reminder

No secrets, no `.env` files, and no real client data in any folder that could reach GitHub. Real pilot folders stay local/private.
