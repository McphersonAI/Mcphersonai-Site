# 08_eval_results — Eval Results

## What belongs here

Eval run outputs (fake-data evals during setup, approved evals before go-live), pass/fail summaries, and eval version references.

## What must NOT be stored here

Edited or truncated eval output; failing evals quietly omitted — failures are recorded and addressed.

## Reminder

No secrets, no `.env` files, and no real client data in any folder that could reach GitHub. Real pilot folders stay local/private.
