# 01_diagnostic — Diagnostic

## What belongs here

The completed Operator Diagnostic for this store: diagnostic notes, findings, dollar-leak observations, and the delivered diagnostic document reference.

## What must NOT be stored here

Raw POS exports or payroll files with employee personal data. Summarize findings; keep raw client systems data in the client's systems.

## Reminder

No secrets, no `.env` files, and no real client data in any folder that could reach GitHub. Real pilot folders stay local/private.
