# Known Limits

This starter pack is deliberately limited. It is:

- **Not a production deployment automation system** — every step is run by a human
- **Not multi-tenant SaaS** — one pilot folder per store, local/private
- **Not customer-facing** — internal use only
- **Not legal approval** — agreements and compliance review happen outside this pack
- **Not a substitute for Blake approval** — completing every checklist still does not equal go-live
- **Live Telegram / MCP / app connection not included yet** — deferred (see `deferred_decisions.md`)
- **Real deployment still requires module-specific setup** — each module's own repo documentation governs its install and run details
